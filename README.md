🔐 Secure Login System

📌 Overview

A secure authentication web application built with Flask implementing strong cybersecurity practices like password hashing, session management, and optional Two-Factor Authentication (2FA).

🚀 Features
User Registration & Login
Secure Password Hashing (Argon2 / bcrypt)
Session Management & Logout
Input Validation
SQL Injection Protection (ORM-based)
CSRF Protection
Optional Two-Factor Authentication (2FA)
🛠 Tech Stack
Python (Flask)
MySQL
Flask-SQLAlchemy
Flask-Login
Flask-WTF
Argon2 / bcrypt
